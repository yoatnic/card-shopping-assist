# 使い方
# カード名,レアリティの順で書いてね
# 1行に付きカード名は1つまで
#
# 例：
# ワイト,Normal
# Ｎｏ.３９ 希望皇ホープ, Ultra
#
# レアリティの書き方
# ノーマル：Normal
# ノーマルレア：N-Rare
# 文字レア：Rare
# スーパーレア：Super
# ウルトラレア：Ultra
# シークレットレア：Secret
# レリーフ：Ultimate
# ホログラフィック：Holographic
# コレクターズレア：Collectors
#
# 「#」で始まる行と空行は無視されるよ

ワイト,Normal
Ｎｏ.３９ 希望皇ホープ, Ultra
